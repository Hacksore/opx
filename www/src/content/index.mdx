Install the `opx` CLI utility with cargo.
```
cargo install opx
```

Create an `.env` file in your project and reference a secret from 1Password.
```shell
APPLE_ID="op://overlayed/apple_id/credential"
```

Run your application to inject the secrets.
```
opx
```
